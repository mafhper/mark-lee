Guia de Contribuição

Para garantir um processo de contribuição suave e eficaz, siga estas diretrizes:

Setup do Ambiente

Siga o guia de "Getting Started" no README do projeto para configurar seu ambiente de desenvolvimento.

Fluxo de Trabalho Git

Crie um branch separado para suas alterações (e.g., `feature/nova-feature` ou `fix/nome-do-bug`). Faça commits atômicos com mensagens claras.

Padrões de Código

Mantenha a consistência com o estilo de código existente. Se for uma nova funcionalidade, inclua testes, se aplicável.
