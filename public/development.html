Arquitetura para a excelência.

Desvendamos os princípios por trás do Mark-Lee: performance, minimalismo e segurança.

Tecnologias que impulsionam.

Rust & Tauri

O coração nativo do Mark-Lee pulsa com Rust, uma linguagem que oferece performance incomparável e segurança de memória, enquanto Tauri nos permite construir um aplicativo desktop leve e eficiente, utilizando o WebView do sistema operacional. Isso resulta em binários pequenos e baixo consumo de recursos.

React 19 & TypeScript

A interface do usuário é construída com a última versão do React e TypeScript. React oferece uma experiência de UI fluida e reativa, enquanto TypeScript garante a robustez e escalabilidade do código frontend com tipagem forte.

Saiba Mais

Vite & TailwindCSS

Para o ambiente de desenvolvimento, usamos Vite para HMR (Hot Module Replacement) ultra-rápido. O TailwindCSS nos permite construir a interface de forma eficiente e modular, com um design consistente e otimizado.

Saiba Mais

WebView2

Componente da Microsoft que renderiza o conteúdo web (interface do Mark-Lee) usando o motor nativo do Edge (Chromium), garantindo performance e integração sem empacotar um navegador inteiro.

Evolução Contínua: Nosso Roadmap.

v1.0.0 - Lançamento Oficial (2025/12/07)

**Novos Temas**: Nord, Synthwave, Forest, Coffee.
**Exportação PDF Profissional**: Layout A4, quebras de página e tipografia limpa.
**Modos de Foco**: Foco (overlay), Imersivo (F11).
**Atalhos Customizáveis**: Defina qualquer tecla para ações.
**Gerenciador de Snippets**: Inserção rápida de tabelas, blocos de código.
**Melhorias de UI/UX**: Arraste de janelas, indicação de modificação, auto-hide de scrollbar e menu bar, auto-save e tempo de leitura.

v0.5.0 - Produtividade (2025/12/07)

**Melhorias no Editor**: Contador de seleção (palavras/caracteres), Modo Foco (spotlight).
**Mais Recursos**: Exportação HTML, Indicador de modificação.
**UX Aprimorada**: Auto-save configurável, Tempo de Leitura, Traduções.

v0.4.0 - Base e Persistência (2025/12/07)

**Novas Funcionalidades**: Persistência de configurações, Arquivos recentes, Suporte a múltiplos formatos de arquivo.
**Melhorias de UI/Core**: Tema Terminal, Auto-hide de menu bar e scrollbar, Transparência de janela, Confirmação de mudanças não salvas, Modo de janela única.

v0.3.0 - Correções e Base (2025/12/07)

**Melhorias UI**: Arrastar janela, ajuste de título, indicadores de modo de visualização.
**Associação de Arquivos**: Correção para abertura via associação de arquivos no Windows.

v0.1.0 - Início do Projeto (2025/12/06)

**Renomeação**: Projeto "Taurim" para "Mark-Lee".

Próxima Fase - Extensibilidade (v1.2.0)

Implementação de um sistema de plugins robusto para permitir que a comunidade estenda as funcionalidades do Mark-Lee com recursos customizados e integrações externas.

Visão Futura - Colaboração (v2.0.0)

Exploração de recursos de sincronização P2P e colaboração em tempo real, com foco em segurança e privacidade dos seus documentos.
