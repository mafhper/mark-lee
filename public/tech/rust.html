Rust & Tauri: Poder e Leveza.

O coração do Mark-Lee pulsa com a performance e a segurança que só Rust pode oferecer, empacotado para o desktop com a eficiência de Tauri.

Por que Rust?

Rust é uma linguagem de programação focada em performance e segurança de memória. Ela nos permite construir um backend para o Mark-Lee que é rápido, confiável e evita classes inteiras de bugs comuns em outras linguagens. Sua promessa de "zero-cost abstractions" significa que você não paga pelo que não usa. **Aplicação no Mark-Lee:** Rust é usado para todas as operações críticas do sistema, como acesso direto ao sistema de arquivos (ler/escrever documentos), gerenciamento de janelas e comunicação com o frontend React, garantindo que o editor responda instantaneamente e de forma robusta.

Por que Tauri?

Tauri é um framework para construir aplicativos desktop multiplataforma usando tecnologias web (HTML, CSS, JavaScript) para a interface do usuário, mas com um backend em Rust. Ao contrário de alternativas como Electron, Tauri utiliza o WebView nativo do sistema operacional (WebView2 no Windows, WebKit no macOS/Linux), resultando em binários significativamente menores e um consumo de RAM muito mais baixo. Para o Mark-Lee, isso significa um instalador de apenas ~3MB e uma experiência mais ágil para o usuário. **Aplicação no Mark-Lee:** Tauri permite que a interface rica do React seja executada em um ambiente nativo, com acesso a APIs do sistema operacional que um aplicativo web comum não teria, como a abertura de diálogos de arquivo nativos ou a gestão de janelas sem bordas.

Exemplo de Código (Rust + Tauri)

Como o frontend (React) invoca uma função Rust para salvar um arquivo:

// Rust (src-tauri/src/main.rs)
#[tauri::command]
async fn save_document(path: String, content: String) -> Result<(), String> {
    std::fs::write(&path, &content).map_err(|e| e.to_string())
}

// TypeScript (frontend - src/services/filesystem.ts)
import { invoke } from '@tauri-apps/api/tauri';

export async function saveFile(path: string, content: string): Promise<void> {
    await invoke('save_document', { path, content });
}

// React Component (src/App.tsx)
// No Mark-Lee, invokamos 'saveFile' quando o usuário salva
const handleSave = async () => {
    // ... lógica para obter path e content ...
    await saveFile(currentPath, markdown);
};

Este exemplo mostra como o frontend em React invoca uma função Rust (`save_document`) via Tauri para realizar operações no sistema de arquivos de forma segura e eficiente, uma integração vital para um editor desktop.
